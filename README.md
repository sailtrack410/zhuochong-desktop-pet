# 桌宠 AI

一只住在 macOS 桌面上的像素猫。它会陪你聊天、提醒你日程、记住你的习惯，时间久了还会对你越来越亲近。

## 它能做什么

**陪伴互动**
- 常驻桌面，可以拖到任意位置
- 单击摸摸头，双击打开控制台
- 右键菜单：喂食、陪玩、让它休息

**AI 对话**
- 在桌宠旁边的输入框直接和它聊天
- 也可以打开控制台面板，查看完整对话记录
- 支持接入 SiliconFlow、DeepSeek 等 AI 服务

**养成系统**
- 好感度、饱食度、精力、健康度四项属性
- 多互动会提升好感，不理它会慢慢下降
- 好感度越高，它说话越热情黏人

**记忆与性格**
- 自动记住你的偏好和习惯
- 记住你们的关系发展历程
- 随着互动增多，性格会慢慢变化

**其他**
- 设定提醒时间，到点它会主动找你
- 明暗模式自动跟随系统
- 驻留在状态栏，不占 Dock 位置

## 安装

1. 前往 [Releases](https://github.com/sailtrack410/zhuochong-desktop-pet/releases) 下载最新版 dmg 文件
2. 双击打开，将「桌宠 AI」拖入「应用程序」文件夹
3. 首次打开时，右键点击应用 → 选择「打开」→ 在弹窗中再次确认「打开」
4. 进入设置页面，填入 AI 服务的 API 地址和密钥

> 首次打开需要右键是因为应用没有经过 Apple 公证，这是正常现象。

## 配置 AI 服务

打开设置页面，需要填写三项：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| API 地址 | AI 服务的接口地址 | `https://api.siliconflow.cn/v1/chat/completions` |
| 模型名称 | 使用的模型 | `Qwen/Qwen3.5-9B` |
| API Key | 你的密钥 | 在对应平台注册获取 |

推荐使用 SiliconFlow，注册即送额度，适合个人使用。

## 系统要求

- macOS 12 及以上
- Apple Silicon（M1 / M2 / M3 / M4）
- 网络连接（AI 对话功能需要）

## 常见问题

**打开时提示"无法验证开发者"**
右键点击应用 → 选择「打开」→ 确认即可。

**桌宠不说话 / 回复异常**
检查设置中的 API 地址和密钥是否正确，确认网络正常。

**数据存在哪里**
所有数据保存在本地，不会上传到任何服务器。

**卸载后数据会丢失吗**
是的，数据存储在本地，删除应用后数据会一起清除。
