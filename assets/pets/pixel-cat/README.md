# Pixel Cat 运行时资源说明

## 目录目的

这套目录用于存放项目实际运行时使用的小猫桌宠资源，不直接依赖原始素材命名。

## 状态映射

- `idle` -> 默认待机
- `sleep` -> 长时间无交互或夜间
- `move` -> 常规移动
- `run` -> 快速移动
- `drag` -> 用户拖拽中
- `click` -> 点击反馈
- `hurt` -> 异常或拒绝反馈
- `hide` -> 靠边停靠

## 资源来源

原始素材目录：

- `../../../Cat_player/gif`

运行时目录中的文件是从原始素材中筛选并重命名得到的。

## 首版前端接入建议

桌宠组件只读取 `manifest.json`，不要在业务代码中硬编码原始素材文件名。

推荐读取流程：

1. 加载 `manifest.json`
2. 根据桌宠业务状态找到 `states[stateName].src`
3. 把该路径渲染到图片组件
4. 如果状态配置了 `nextState`，则在短反馈结束后切回对应状态

## 说明

当前 `chat` 没有独立说话动画，默认回退到 `idle`。
